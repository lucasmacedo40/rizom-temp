# Portal Web ESP8266 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adicionar portal web completo ao firmware Wemos D1 Mini — acessível via IP na rede local após provisionamento, protegido por login com cookie.

**Architecture:** O `ESP8266WebServer portal(80)` existente é reutilizado em dois modos: AP (provisioning, comportamento atual) e STA (portal completo). HTMLs ficam no LittleFS (`firmware/data/`), servidos via `serveStatic`. APIs JSON em `/api/*` alimentam as páginas via `fetch()`. Sessão por cookie simples com token em RAM.

**Tech Stack:** C++ Arduino ESP8266, LittleFS (core 3.x nativo), ESP8266WebServer, EEPROM, HTML/CSS/JS vanilla, PlatformIO (espressif8266)

## Task 3: Adicionar buffer circular de histórico de temperatura

**Files:**
- Modify: `firmware/rizom_temp_WEMOS_D1.ino` (após a seção de globals, antes de `void salvarConfig()`)

- [ ] **Step 1: Adicionar constante, structs e variáveis globais de histórico após a linha `String deviceMac;`**

```cpp
// ─── Histórico de temperatura em RAM ──────────────────────────
#define HIST_SIZE 20

struct Leitura { unsigned long ts; float t; };
Leitura historico[HIST_SIZE];
int histCount = 0, histHead = 0;

void pushLeitura(float temp) {
  historico[histHead] = {millis(), temp};
  histHead = (histHead + 1) % HIST_SIZE;
  if (histCount < HIST_SIZE) histCount++;
}

String histJson() {
  String j = "[";
  int start = (histCount < HIST_SIZE) ? 0 : histHead;
  for (int i = 0; i < histCount; i++) {
    int idx = (start + i) % HIST_SIZE;
    if (i > 0) j += ",";
    // age em segundos desde a leitura (relativo ao device, não epoch)
    unsigned long age = (millis() - historico[idx].ts) / 1000;
    j += "{\"age\":" + String(age) + ",\"t\":" + String(historico[idx].t, 2) + "}";
  }
  return j + "]";
}
```

// ─── Sessão e helpers do portal ───────────────────────────────
String gerarToken() {
  String t = "";
  for (int i = 0; i < 32; i++) t += String(random(16), HEX);
  return t;
}

void servirArquivo(const String& path, const String& ct) {
  if (!LittleFS.exists(path)) { portal.send(404, "text/plain", "Not found"); return; }
  File f = LittleFS.open(path, "r");
  portal.streamFile(f, ct);
  f.close();
}

bool verificarSessao() {
  if (sessionToken.length() == 0) {
    portal.sendHeader("Location", "/login");
    portal.send(302, "text/plain", "");
    return false;
  }
  String cookie = portal.header("Cookie");
  if (cookie.indexOf("sid=" + sessionToken) >= 0) return true;
  portal.sendHeader("Location", "/login");
  portal.send(302, "text/plain", "");
  return false;
}

// ═══════════════════════════════════════════════════════════════
//  PORTAL STA — Portal completo (modo conectado)
// ═══════════════════════════════════════════════════════════════
void iniciarPortalSTA() {
  const char* hdrs[] = {"Cookie"};
  portal.collectHeaders(hdrs, 1);

  portal.serveStatic("/style.css", LittleFS, "/style.css");

  portal.on("/", HTTP_GET, [] {
    portal.sendHeader("Location", sessionToken.length() ? "/status" : "/login");
    portal.send(302, "text/plain", "");
  });

  portal.onNotFound([] { portal.send(404, "text/plain", "Not found"); });

  portal.begin();
  Serial.println("[Portal] Portal STA iniciado na porta 80");
}
